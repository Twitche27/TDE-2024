# 📊 Data Science Applied to the Comparative Analysis of HDI and Teenage Pregnancy in Brazil

A data science project that investigates the relationship between the **Human Development Index (HDI)** and **teenage pregnancy rates** across Brazilian states between **2014 and 2021**, using public datasets from **IBGE** and **DATASUS**.

> University research project developed for the Computer Science program at Universidade Católica de Santos (2024). :contentReference[oaicite:0]{index=0}

## 📖 About

This project explores whether socioeconomic indicators—especially the **Human Development Index (HDI)**—are associated with teenage pregnancy rates in Brazil.

The study analyzes data from **2014 to 2021**, comparing the most populous state from each Brazilian region:

- North — Pará
- Northeast — Bahia
- Midwest — Goiás
- Southeast — São Paulo
- South — Paraná

The project combines data collection, cleaning, automation, and visualization to identify trends and possible correlations.

---

## ✨ Features

- Collection of public datasets
- Automated data extraction using Selenium
- Data cleaning and preprocessing with Pandas
- Comparative analysis between HDI, household income, and teenage pregnancy
- Data visualization through charts
- Research documentation and findings

---

## 🛠 Technologies

- Python
- Pandas
- Selenium
- Regular Expressions (re)
- Tkinter
- CSV

---

## 📂 Data Sources

The project uses publicly available datasets from:

- **IBGE** – Human Development Index (HDI)
- **IBGE News Portal** – Household Income
- **DATASUS / SINASC** – Live births by mother's age

---

## 📈 Project Goal

The objective is to investigate whether regions with lower socioeconomic development tend to present higher rates of teenage pregnancy and identify trends over time.

## 📖 Sobre

Este projeto aplica técnicas de Ciência de Dados para investigar a relação entre o **Índice de Desenvolvimento Humano (IDH)** e os índices de **gravidez na adolescência** no Brasil.

Foram analisados dados públicos referentes ao período de **2014 a 2021**, utilizando como amostra os estados mais populosos de cada região brasileira:

- Norte — Pará
- Nordeste — Bahia
- Centro-Oeste — Goiás
- Sudeste — São Paulo
- Sul — Paraná

O projeto envolve coleta automatizada de dados, tratamento, análise e visualização das informações para identificar possíveis correlações.

---

## ✨ Funcionalidades

- Coleta de dados públicos
- Extração automatizada utilizando Selenium
- Limpeza e transformação de dados com Pandas
- Comparação entre IDH, renda domiciliar e gravidez precoce
- Geração de gráficos para análise
- Documentação da pesquisa

---

## 🛠 Tecnologias

- Python
- Pandas
- Selenium
- Expressões Regulares (re)
- Tkinter
- CSV

## 📂 Fontes de Dados

Os dados utilizados foram obtidos em bases públicas:

- **IBGE** – Índice de Desenvolvimento Humano (IDH)
- **Portal de Notícias do IBGE** – Renda domiciliar per capita
- **DATASUS / SINASC** – Nascidos vivos por faixa etária da mãe

---

## 📈 Objetivo

Investigar se estados com menor desenvolvimento socioeconômico apresentam maiores índices de gravidez na adolescência, analisando tendências ao longo dos anos.
